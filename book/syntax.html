Syntax Reference

A quick guide to Fax-lang keywords, types, and structures.

Basic Syntax

Every Fax program starts with a main function.

fn main() {
    println("Hello, World!");
}

Variables

Use let for immutability and let mut for mutable variables.

let x = 10;          // Immutable
let mut y = 20;      // Mutable
y = y + x;

State Machines

state_machine Device {
    state Off {
        fn turn_on() -> On { ... }
    }
    state On {
        fn turn_off() -> Off { ... }
    }
}

Memory Life-Force

Variables decay over time based on their usage.

let x = 100;
// Read costs 0.02 Life-Force
let y = x; 
// x now has 0.98 Life-Force remaining
For the full 3000-line specification, please refer to the technical manual.
